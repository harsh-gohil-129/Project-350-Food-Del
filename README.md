## üõ† What We Built

**Project-350** is a full-featured food ordering website and app powered by **React JS, Node.js, Express.js, MongoDB, and Stripe**.
This platform includes:

‚úÖ A modern and responsive **user frontend**
‚úÖ A **backend API server** for business logic
‚úÖ An **admin panel** for managing food items, customer orders, payments, and delivery statuses



## ‚öô Core Technologies

* **Frontend**: React JS (with Context API for state management)

* **Authentication**: JSON Web Token (JWT)
* **Payments**: Stripe API Integration



## ‚ú® Key Features

* Modern, clean, and responsive UI

* Add to Cart and Cart Management

* Safe payments with **Stripe**
